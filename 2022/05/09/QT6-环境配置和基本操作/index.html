QT6: 环境配置和基本操作

文本基于macOS aarch64

1. 安装

使用 homebrew 可以非常容易地完成QT的安装：

2. 配置 Kit

因为架构上的不同，导致我们必须使用 $6.2$ 及更高的QT版本，并且需要在 qt-creator 中自行对应配置 qmake 的地址。

在 Preferences -> Qt Versions -> Add 中打开文件夹，然后找到 Macintosh HD/opt/homebrew/bin/qmake，点击确认。这样就可以自动检索到对应的 Apple Clang 编译器。

⚠️如果没看到 opt 文件夹，这是因为opt文件夹被系统识别成隐藏文件，默认不会显示，这时按 command + shift + . 就可以显示隐藏文件。同理，bin 目录下找 qmake 时也是一样的。

最后在 Kits 中点击 Apple-64bit 进行配置，修改 Qt version和编译器。这样就可以正常使用了。

3. 简单操作

  1. *.ui 文件可以用 Designer 模块进行可视化编程。
  2. 对组件右键，选择 slot，可以转入到对应的操作函数，可以直接进行编程。
  3. qt-creator的mac版本体验很接近 Xcode，还可以配置vim模式
