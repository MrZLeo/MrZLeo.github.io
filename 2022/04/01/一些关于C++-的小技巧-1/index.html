3. void main() 是否合理？

不合理。

永远不应该使用 void main。标准的 C/C++ 程序都需要返回 int 给调用他的系统。没有办法处理返回值的情况下，返回值会被忽略（例如嵌入式或操作系统的场景下），但不代表 void main 符合标准。
